課程一：3D童玩鬥片

| 時間 | 主題 | 概述 |
| :---- | :---- | :---- |
| 30min | 認識MagicaVoxel | 使用MagicaVoxel建模軟體，熟悉基本功能操作。 |
| 30min | 童玩鬥片建模 | 按步驟設計鬥片基本外型，並加入個人創意元素。 |
| 30min | Cura模型切片 | 使用Cura軟體進行切片，認識列印參數。 |
| 30min | 3D列印實作 | 操作3D列印機，印製童玩鬥片作品。 |
| 30min | Voxel 動畫設計 | 使用CubicCode平台，製作霓虹名牌動畫。 |
| 30min | 自由發揮 | 使用特殊麥克筆進行作品彩繪塗色，或設自由設計其他作品。 |
