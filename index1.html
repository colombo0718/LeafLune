🌙 品牌起源

葉與月，是一段早就注定的宇宙。

LeafLune 不是設計出來的名字。它是兩個人的相遇，在不知不覺中，長成了一個完整的教育宇宙。

🌿 Leaf · 葉
她 · Xuân
越南語 Xuân = 春天

春天讓種子萌芽，讓葉子長出。她代表的是教育、成長與啟蒙，是所有學習的根基所在。

名字帶來了 Leaf（葉），Leaf 長成了 宣學習。

Leaf → 宣學習 → 地 · 樂 · 教育
🌕 Lune · 月
你 · 太空
Space 是策略，Lune 是月亮

月亮是太空的守望者，也是策略的象徵。代表技術、系統與競技，是對戰與挑戰的最高舞台。

名字帶來了 Lune（月），Lune 成為了 策略空間。

Lune → 策略空間 → 天 · 比 · 競賽
Leaf + Lune = LeafLune 宇宙

🎮 GameGeek 深度介紹

不只是遊戲，是 AI 遊戲宇宙的閘道。

GameGeek（GG）是通用遊戲介面（Universal Game Interface），四個階段讓人類、RL、LLM 與腳本共存在同一個 AI 遊戲生態平台。

🥇

Phase 1 · 遊戲控制殼（Game Shell）

共用直覺的 Web 遊戲主機 UI。D-pad、功能鍵、中央遊戲 iframe——任何支援協定的遊戲都能上架，介面自動一致。

Web UI iframe D-pad
🥈

Phase 2 · AI 控制介面（AI Agent Integration）

GG 成為遊戲中介，提供標準化協定（state / action / reward）。可掛載 RL Agent、LLM Agent、腳本 Bot 或人類玩家。

RL Agent LLM Agent state/action/reward
🥉

Phase 3 · Blockly 腳本系統（GG Script）

內建 Blockly 讓使用者編寫自動化腳本、製作 rule-based AI，讓非工程背景的人也能製作 Bot 並理解 AI 決策流程。

Blockly rule-based AI 視覺化腳本
🏆

Phase 4 · WebRTC 連線大廳（GG Online）

所有遊戲共用的多人連線系統，提供房間建立、玩家配對、觀戰模式，支援 AI vs 人類、AI vs AI 等多種對戰形式。

WebRTC 匹配系統 觀戰模式
